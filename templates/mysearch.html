New version of Australian Trade Mark Search

IP Australia’s upgrade to trade mark systems is now complete.

This is the new version of Australian Trade Mark Search.

For information about the changes to Australian Trade Mark Search: click here.

Please also refer to the system’s Help for updated information about search functionality, including more detailed Release Notes.

The upgrade included the largest ever data migration in IP Australia’s history with hundreds of millions of records migrated to a new system. Please see the system’s About page for more information, including the updated Quality Declaration in relation to data.
